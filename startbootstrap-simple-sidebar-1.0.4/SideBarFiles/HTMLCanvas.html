HTML5 Canvas

ele elemento <canvas> se usa para dibujar algo en una página web

Podemos dibujar rectángulos, círculos, etc.

Estos elementos se dibujan via Javascript, el <canvas> es solo un contenedor par gráficos, se debe uar javascript para dibujar dichos gráficos.

Veamos como se ve

Hola mundo

Como podemos ver hermos dibujado un rectángulo, este elemento me permite definir el ancho y el largo del rectángulo.

Ahora dibujemos un circulo dentro de un canvas

Your browser does not support the HTML5 canvas tag.

Vemos el código 

                            <canvas id="myCanvas1" width="200" height="100" style="border:1px solid #d3d3d3;">
                            hola.</canvas>

                            <script>
                            var c = document.getElementById("myCanvas1");
                            var ctx = c.getContext("2d");
                            ctx.beginPath();
                            ctx.arc(95,50,40,0,2*Math.PI);
                            ctx.stroke();
                            </script>
