Element Form de HTML

Aquí describimos los más importantes elementos HTML

El elemento <input>

Este elemento tiene muchas variaciones.

El elemento <select>

Este elemento define una lista desplegable.

Veamos como se ve

Su código

                            <select name="carros">
                            <option value="volvo">Volvo</option>
                            <option value="saab">Saab</option>
                            <option value="fiat">Fiat</option>
                            <option value="audi">Audi</option>

El elemento <option> define las opciones que se pueden seleccionar. Usualmente la lista muestra el primer elemento.

El elemento <textarea>

Veamos como se ve

Su código

                            <textarea name="mensaje" rows="10" cols="30">
                            Este es un text area bastante grande.
                            </textarea>

El elemento <button>

Asi es como se ve.

Su código

                            <button type="button" onclick="alert('Hola!')">Click!</button>

Elementos HTML

Tag Descripción
<form> Define la forme de ingreso por el usuario
<input> Define un control de ingreso
<textarea> Define un control de ingreso de varias líneas
<label> Define una etiqueta para un elemento <input>
<fielset> Agrupa elementos relacionados en un form
<legend> Define un caption para un elemento <fieldset>
<select> Define una lista desplegable
<optgroup> Define un grupo de opciones relacionadas en una lista desplegable
<option> Define una opción en una lista desplegable
<button> Define un boton clickeable
datalist Especifica una lista de opciones predefinidas para controles de ingreso
<keygen> Define un campo generador de keys
<output> Define el resultado de un calculo
