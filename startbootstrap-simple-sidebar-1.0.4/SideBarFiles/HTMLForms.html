HTML Forms

El elemento <form>

Estos elementos son usados para tomar ingresos de datos del usuario

El elemento <input>

Este elemento es el mas importante de los <form>. Tiene variaciones dependiendo del tipo de atributo del elemento. Veamos algunos tipos:

Tipo Descripción
text Tipo de ingreso de texto común
radio define el tipo de ingreso de radio button, se puede seleccionar una o varias
submit Define un botón de confirmación
