Elementos de HTML5

Tag Descripción
<article> Define un artículo
<aside> Define contenido al costado del contenido de una página
<audio> Define contenido audible
<bdi> Aisla una parte de un texto que podria estar formateado en una manera diferente de otro texto externo
<canvas> Se usa para dibujar gráficos via javascript
<datalist> Especifica una lista de opciones pre-definida para controles de ingreso
<details> Define detallesa dicionales que el usuario puede escoder o mostrar
<dialog> Define un cuadro de diálogo en una ventana
<embed> Define un contenedor para una aplicación externa no HTML
<figcaption> Define un caption para un elemento <figure>
<figure> Especifica contenido automático
<footer> Define un pie de página para una sección
<header> Define una cabecera para una sección de un documento
<keygen> Define un generador de key para elementos <form>
<main> Especifica el contenido principal de un documento
<mark> Define texto marcado o enfatizado
<menuitem> Define un item de jun menu que el usuario puede invocar via pop-up
<meter> Define una medida escalar con un rango conocido
<nav> Define links de navegación
<output> Define el resultado de un calculo
<progress> Representa el progreso de una tarea
<rp> Define que mostrar en los navegadores que no soportan anotaciones en ruby
<rt> Define una explicación de caracteres para tipografía asiatica
<ruby> Define una anotación en ruby
<section> Define una sección en un documento
<source> Define multiples recursos multimedia para tags como <video> y <audio>
<sumary> Define un encabezamiento visible para elementos <details>
<time> Define la fecha o la hora
<track> Define tracks de texto para elementos <video> y <audio>
<video> Define un video o película
<wbr> Define un posible quiebre de página
