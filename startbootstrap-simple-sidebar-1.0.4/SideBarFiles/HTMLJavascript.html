Scripts en HTML

los Javascripts hacen a las páginas HTML más dinámicas e interactivas

El tag <script>

El tag <script> es usado del lado del cliente, El elemento <script> puede contener sentencias javascript o a su vez llamar a un archivo .js externo.

Miren como se ve

Javascript puede cambiar el contenido de un archivo HTML 

                          
                          
Cambia este contenido

Veamos el código 

                          
                            <button type="button" onclick="myFunction()">Click!</button>
                            <p id="demo">Cambia este contenido</p>
                            <script>
                            function myFunction() {
                              document.getElementById("demo").innerHTML = "Contenido cambiado";
                            }

Tags de scripts de HTML

Tag Descripción
<script> Define un script del cliente
<noscript> Define un contenido alternativo para usuarios que no soportan scripts del lado del cliente
