Elementos HTML

los Documentos HTML están hechos de elementos HTML

Estos elementos son escritos con un tag de inicio, luego el contenido y después el tag de fin.

<nombre del tag>contenido del tag</nombre del tag>

Ejemplo

<p>Este contenido pertenece a un parrafo</p>

Tabla eplicativa

Tag de inicio Contenido Tag de fin
<h1> Este es una cabecera </h1>
<p> Este es un parrafo </p>

Elementos anidados

Esto quiere decir que podemos poner elementos dentro de otros elementos

Ejemplo

                    <html>
                    <body>
                    <h1>Cabecera</h1>
                    <p>Parrafo</p>
                    </body>
                    </html>
